# 不可变类

---

## 场景
一些不可变值的类描述


## 示例
~~~java
/**
 * 第十七章
 * 不可变性
 * 指一些不可变类,类的参数域和类都要加final
 */
public final class MinChange {

    private final String name;

    private final int age;

    public MinChange(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String name(){
        return name;
    }

    public int age(){
        return age;
    }
}

~~~
