# 重写compare方法

---


## 场景
~~~text
一些需要比较的类
~~~

## 示例
~~~java
/**
 * 第十四章:
 * 比如这个.
 * 对于一个枚举等在对象性质上明显需要排序方法的时候,可以考虑重写compare方法.
 * 对于jdk1.8而言,可以考虑引入1.8的比较器方法,静态导入来进行比较
 */
public class Compare implements Comparable<Compare>{

    private int a;

    private int b;

    private int c;

    public Compare() {
    }

    public Compare(int a, int b, int c) {
        this.a = a;
        this.b = b;
        this.c = c;
    }

    public int getA() {
        return a;
    }

    public void setA(int a) {
        this.a = a;
    }

    public int getB() {
        return b;
    }

    public void setB(int b) {
        this.b = b;
    }

    public int getC() {
        return c;
    }

    public void setC(int c) {
        this.c = c;
    }

    @Override
    public String toString() {
        return "Compare{" +
                "a=" + a +
                ", b=" + b +
                ", c=" + c +
                '}';
    }

    private static Comparator<Compare> comparator = comparingInt((Compare p) -> p.a )
            .thenComparingInt((Compare p) -> p.b)
            .thenComparingInt((Compare p) -> p.c);

    @Override
    public int compareTo(Compare o) {
        return comparator.compare(this , o);
    }
}
~~~
